\begin{quote}
\small\oxdoc~is a small software package for generating documentation in 
HTML format from comments in {\tt ox} source code. It is inspired by
Sun Microsystems' Javadoc. 
To use \oxdoc, the user needs to write comments in {\tt ox} source code
in a special format 
and run the \oxdoc~application to extract these comments and
generate a comprehensive HTML (web page) document of the available 
functions and classes in the original code. \bigskip

\end{quote}
